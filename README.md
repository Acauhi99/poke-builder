# Pokebuilder

## Descrição

Pokebuilder é uma aplicação web que permite aos usuários criar um time de 6 Pokémons. Cada Pokémon pode ser selecionado de uma lista, e informações detalhadas sobre cada Pokémon são exibidas ao selecioná-lo. Este projeto é construído usando JavaScript puro, HTML e CSS.

## Funcionalidades

- Seleção de até 6 Pokémons para o time.
- Exibição de informações detalhadas de cada Pokémon selecionado.
- Interface de usuário responsiva e amigável.

## Tecnologias Utilizadas

- JavaScript
- HTML
- CSS
- PokeAPI (para obter dados dos Pokémons)
